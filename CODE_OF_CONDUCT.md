# 行为准则

本项目仅供学习与测试。任何使用都必须符合法律法规和问卷平台的服务条款。

## 适用范围
- 贡献者、提交 Issue/PR 的社区成员。
- 使用二进制或源码运行本工具的任何人。

## 核心原则
1. **合法合规**：不得用于任何违法违规场景。
2. **诚实科研**：严禁伪造、篡改或批量生成虚假学术/实验数据。
3. **尊重问卷**：严禁恶意刷问卷、污染他人数据、干扰正常调研。
4. **授权优先**：仅在获得问卷创建者明确许可的前提下使用。

## 明确禁止的行为
- 利用本工具规避问卷平台的验证、计数或反作弊措施。
- 批量提交虚假答卷、刷票、刷量或其他流量造假。
- 任何形式的网络攻击、入侵、或对第三方造成损害的行为。
- 未经许可收集、存储、传播他人个人信息或敏感数据。

## 举报与处理
- 发现违规使用或侵权，请发送邮件至 `mail@hungrym0.top` 举报或申请删除。
- 维护者可在不事先通知的情况下拒绝支持、关闭相关 Issue/PR，必要时配合平台或执法机构。

## 社区沟通
- 技术讨论与变更：GitHub Issues/PR。
- 使用反馈与新功能建议：QQ群（见 README）。

参与或使用本项目即表示同意并遵守本准则。
